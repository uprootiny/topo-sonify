You've learned the five core ideas:

1. Points are voices (β₀).

2. Connections merge them (Vietoris-Rips).

3. Loops add modulation (β₁).

4. Features are born and die (persistence → delay).

5. Curvature shapes the room (χ → reverb).

The instrument panel wires all of this through the Rust engine via WASM: BettiDrone, SimplicialPluck, MorseSynth → BettiDistortion → PersistenceDelay → CurvatureReverb.
